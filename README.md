# AlltalkRoundToggle
Plugin de CS2 para mod "CounterStrikeSharp" que brinda la función de hablar entre todos al finalizar la ronda.
Permite hablar con todos los participantes del mapa al finalizar una ronda, estando muertos o vivos. Al comenzar la siguiente ronda, automáticamente se mutea la función otra vez.
-----------------
# Para integrarlo
Para integrarlo como plugin en tu server hosteado, debes ubicar la carpeta de cssharp en tu host, ubicar la carpeta de plugins del mismo mod y crear una nueva carpeta llamada como el plugin junto a tus otros plugins.
Lo que debes pegar en el servidor es: bin/Release/net8.0/AlltalkRoundToggle.dll
Luego reiniciar el server y probar su funcionamiento.
